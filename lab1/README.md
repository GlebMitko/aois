# Лабораторная работа №1 — Представление чисел в памяти компьютера

Проект выполнен по условиям файла задания: переводы в прямой/обратный/дополнительный коды, сложение и вычитание в дополнительном коде, умножение и деление в прямом коде, операции с плавающей точкой IEEE-754 (32 бита) и BCD-сложение для варианта A.

## Что умеет программа

- переводить десятичное целое число в:
  - прямой код;
  - обратный код;
  - дополнительный код;
- складывать два целых числа в дополнительном коде;
- вычитать через отрицание вычитаемого и сложение;
- умножать два целых числа в прямом коде;
- делить два целых числа в прямом коде;
- работать с вещественными числами в формате IEEE-754 single precision (32 бита):
  - перевод,
  - сложение,
  - вычитание,
  - умножение,
  - деление;
- складывать числа в BCD.

## Вариант A

По умолчанию вариант A реализован как **8421 BCD**.
Дополнительно в интерфейсе доступны и остальные коды:
- 2421 BCD
- 5421 BCD
- Excess-3
- Gray BCD

## Структура проекта

- `main.py` — точка входа
- `app/controller.py` — контроллер
- `app/model/` — вся бизнес-логика
- `app/view/` — графический интерфейс на `tkinter`
- `tests/` — unit-тесты
- `run_tests.py` — запуск тестов
- `run_coverage.py` — запуск тестов с отчётом покрытия
- `coverage_report.txt` — готовый отчёт покрытия

## Запуск в PyCharm

1. Открой папку проекта `lab1_project` как проект.
2. Убедись, что используется Python 3.11+.
3. Запусти файл `main.py`.

## Запуск из корня проекта

### Приложение

```bash
python main.py
```

### Тесты

```bash
python run_tests.py
```

### Покрытие

```bash
python run_coverage.py
```

## Зависимости

Основное приложение использует только стандартную библиотеку Python.
Для отдельного запуска отчёта покрытия нужен только dev-пакет:

```bash
pip install -r requirements-dev.txt
```

## Примечания

- Ввод вещественных чисел поддерживается во вкладке IEEE-754.
- Для интерфейса используется `tkinter`, поэтому программа нормально запускается в PyCharm на обычной локальной машине.
- Все ключевые вычисления вынесены из UI в отдельные модули модели.
